👋 WELCOME!

You have a complete, production-ready Go application that analyzes
Ethereum smart contract function calls and their calldata sizes.


⚡ QUICK START (Total time: ~20 minutes)

1. Get Etherscan API key (5 min)
   → Visit: https://etherscan.io/apis
   → Create account / sign in
   → Generate API key
   → Copy it

2. Create config file (5 min)
   → Run: cp config.example.json config.json
   → Edit config.json
   → Paste your API key
   → Add your contract addresses and function selectors

3. Find function selectors (5 min)
   → Visit: https://www.4byte.directory/
   → Search for your functions
   → Copy the 4-byte selectors

4. Run analysis (<1 min)
   → ./eth-analysis -config config.json -output text
   → See your results!

🎯 WHAT IT DOES

1. Reads your contract list (addresses + function selectors)
2. Fetches transaction data from Etherscan API
3. Filters by function and extracts calldata sizes
4. Calculates statistics:
   • Count (number of calls)
   • Min/Max (smallest/largest calldata)
   • Mean (average size)
   • Median (middle value)
   • Mode (most common size)
5. Outputs results (JSON, CSV, or text table)

Time to analyze: ~5-60 seconds (depends on contracts & time range)
